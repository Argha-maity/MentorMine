# MentorMine
This is an AI coding mentor designed to help developers level up.

## Project Setup
This project is built using **React + Vite**, providing a minimal setup with HMR and ESLint.

### Available Plugins
- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react) (Babel/Oxc)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) (SWC)

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
