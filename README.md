MentorMine: AI-Powered Coding Mentor
MentorMine is a sophisticated Chrome Extension designed to help developers solve LeetCode problems through progressive hinting and context-aware guidance. Unlike generic AI tools, MentorMine analyzes the user's specific code, current LeetCode error messages, and problem constraints to provide "Socratic" mentoring—leading the user to the solution without giving it away.

🚀 Key Features
    Context-Aware Hints: Automatically scrapes LeetCode problem descriptions and error logs (TLE, Wrong Answer, etc.) to provide relevant advice.

    Progressive Hinting System: Three levels of guidance—Nudge, Direction, and Approach—to prevent solution spoilers.

    Real-Time Code Analysis: Monitors the LeetCode editor to provide feedback on your current logic.

    Smart Chat Interface: A focused coding assistant that uses Socratic questioning to build problem-solving skills.

    Professional UI/UX: Built with React and Vite, featuring a sleek side-panel interface with full Markdown and Syntax Highlighting support.

🛠️ Tech Stack
    1. Frontend: React.js, Tailwind CSS (or your styling choice)
    2. Bundler: Vite
    3. AI Engine: Groq API (Llama 3.3 70B Model)
    4. Extension APIs: Chrome SidePanel API, Storage API, Content Scripts, Background Service Workers
    4. Parsing: React-Markdown, React-Syntax-Highlighter

🏗️ Architecture
MentorMine utilizes a decoupled architecture to ensure performance and reliability:
    1. Content Script: High-efficiency DOM scraper that monitors LeetCode state changes and editor inputs.
    2. Background Service Worker: Orchestrates communication between the browser and the Groq LLM while managing API keys via environment variables.
    3.Side Panel (React): A responsive UI that displays AI responses and manages user session history.

⚙️ Installation & Setup
    Prerequisites
        Node.js (v18+)
        A Groq API Key

    Step-by-Step Installation
        1. Clone the repository:
            Bash
            git clone https://github.com/yourusername/MentorMine.git
            cd MentorMine

        2. Install dependencies:
            Bash
            npm install

        3. Set up environment variables:
            Create a .env file in the root directory:
                VITE_GROQ_KEY=your_groq_api_key_here

        4. Build the project:
            Bash
            npm run build
        5. Load the extension in Chrome:
            Open Chrome and navigate to chrome://extensions/.
            Enable Developer mode (top right).
            Click Load unpacked and select the dist folder generated by the build.
